CNVTR

Turn any public link into studio-ready audio

Download

Free desktop app for Windows and macOS. No account required. Runs entirely on your computer.

Download for Windows Download for Mac

Get the latest installers from GitHub Releases. Windows: run the .exe installer. Mac: open the .dmg and drag CNVTR to Applications.

What it does

Paste links from YouTube, Instagram, X (Twitter), TikTok, SoundCloud, Vimeo, or Facebook. Choose MP3 (up to 320 kbps) or WAV. Batch convert, add metadata and album art, pick your output folder. Built for DJs, producers, and editors.

Requirements

CNVTR includes yt-dlp. You need FFmpeg on your system for conversion:

  • Windows: Download FFmpeg, extract, and add the bin folder to your system PATH.
  • Mac: brew install ffmpeg (or download from ffmpeg.org).
